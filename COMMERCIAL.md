# Commercial and Pro Edition

Lobster AI System has a public Community edition and a private Pro edition.

## Community Edition

The public repository remains open-source and useful for developers who want a local-first Python runtime repair primitive:

- Run a failing command and capture the real error.
- Classify common runtime failures.
- Suggest small safe fixes.
- Apply conservative local repairs with `--apply`.
- Emit machine-readable JSON reports.
- Keep source attribution visible in shared reports and demos.

## Pro Edition

The Pro edition is private and contains the higher-value commercial roadmap:

- Advanced repair rule packs.
- Multi-file project repair planning.
- Batch repository scanning.
- CI and agent workflow adapters.
- HTML / visual reports.
- Private benchmarks and real-world failure corpus.
- Team policy controls and commercial support workflows.

## Why This Split Exists

The Community edition is for trust, demos, learning, and early adoption. The Pro edition protects the deeper product work from low-attribution cloning while keeping the public project alive as the canonical source.

If you are interested in commercial usage, collaboration, or the Pro roadmap, open an issue in the public repository and reference "Lobster Pro".
